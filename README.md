# Bank Loan Analysis

## Objective
To analyze customer loan data and identify high-risk borrower segments and loan patterns.

## Tools & Technologies
- Excel
- SQL
- Power BI
- Tableau
- Python (Pandas)

## Dataset
The dataset contains customer loan details including loan amount, interest rate, grade, sub-grade, and employment information.

## Key Insights
- Identified high-risk sub-grade categories (B3, C5, D2)
- Analyzed relationship between interest rate and loan grade
- Studied revolving balance and repayment trends
- Observed geographic distribution of loan defaults

## Dashboards
Interactive dashboards created using Power BI and Tableau for business decision-making.

## Outcome
This project demonstrates practical skills in data cleaning, visualization, and analytical reporting for real-world financial data.
